# Спецификация дизайна (DESIGN.md)

Данный документ описывает правила оформления и визуальный стиль сайта `.md parser`. Все последующие изменения интерфейса и новые страницы должны строго соответствовать этим правилам для сохранения единого стиля «старых японских сайтов».

---

## 1. Цветовая палитра (Color Palette)

* **Фон страницы:** `#000000` (абсолютно черный).
* **Основной текст:** `#FFFFFF` (абсолютно белый).
* **Название сайта:** `#BC002D` (темно-красный, цвет солнца на флаге Японии).
* **Ссылки навигации:** `#FF00FF` (ярко-розовый / фуксия).
* **Границы и рамки:** `#FFFFFF` (тонкие белые рамки `1px solid #FFFFFF` для ретро-дизайна).

---

## 2. Типографика (Typography)

Для достижения эффекта старых операционных систем и пиксельного рендеринга используется шрифт **MS Gothic**:

```css
font-family: "MS Gothic", "MS ゴシック", monospace;
font-size: 16px; /* Оптимальный размер для пиксельного эффекта */
font-smooth: never;
-webkit-font-smoothing: none;
-moz-osx-font-smoothing: none;
```

* **Особенности:** Сглаживание шрифтов должно быть полностью отключено. Текст должен выглядеть слегка пиксельным и резким.

---

## 3. Макет и навигация (Layout & Navigation)

* **Боковая панель (Sidebar):**
  * Находится слева.
  * Открывается и закрывается мгновенно, **без каких-либо плавных анимаций выдвижения**.
  * Переключатели (кнопки):
    * Для закрытия: просто текст `<-` (без иконок).
    * Для открытия: просто текст `->` (без иконок).
  * Цвет ссылок в меню: `#FF00FF` (фуксия).
  * Переход по ссылкам осуществляет мгновенную смену раутов.
* **Адаптивность:**
  * На мобильных устройствах боковая панель при открытии перекрывает контент или располагается поверх него с рамкой, сохраняя функциональность и ретро-вид.

---

## 4. Интерактивные элементы (Interactive Elements)

### Название сайта
* Текст по умолчанию: `.md parser`
* Цвет: `#BC002D`
* При наведении курсора (hover) текст меняется на `.md パーサー`.

### Кнопка "Parse"
* Текст по умолчанию: `Parse`
* При наведении курсора (hover) текст меняется на `パース`.
* **Важное условие:** Размер кнопки не должен изменяться при смене текста (кнопка имеет фиксированный размер или рассчитывается по максимальной ширине обоих слов с помощью CSS Grid, чтобы избежать сдвигов верстки).
* Оформление: черная кнопка с белой рамкой `1px solid #FFFFFF`, белым текстом и резким ховером (например, инверсия цветов при наведении).

---

## 5. Общие принципы верстки

1. Никаких скруглений углов (`border-radius: 0`).
2. Никаких градиентов, размытий, теней (box-shadow) и плавных CSS-переходов (transition).
3. Границы таблиц, блоков, кнопок и инпутов оформляются в стиле простых табличных рамок (`1px solid #FFFFFF`).
